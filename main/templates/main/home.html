Welcome!

Hi! I’m Avigyan Chakraborty, an undergraduate in the B.Stat (Hons) program at ISI Kolkata, Class of 2028. This is my personal website—a curated collection of academic notes, personal reflections (both academic and non-academic), and a growing archive of typesetting and grammatical pet peeves.

Explore

  • Calculus — A collection of articles and problem sets bridging high school calculus and real analysis.
  • Olympiads — References and selected materials I used while preparing for Math Olympiads in high school.
  • Blog — Reflections on life and other topics, mostly non-academic in nature.
  • Archive — Mostly math and academic posts.
  • Pet Peeves — Nit-picky thoughts on code formatting, typesetting, and grammar mistakes I’m obsessed with.
  • Contact — Links to my social media and other ways to reach me.

My Projects

  • This Website — Built from scratch using Django and Markdown. Features include static pages, a blog system, and math rendering—all integrated with the elegance of my favorite color scheme, Tokyo Night.
  • BON — A problem database management system built entirely in Python for storing LaTeX-formatted content. It also supports integration with LaTeX documents. A more detailed description of the project is available in the README file.
  • Custom LaTeX Styles — A custom style file for typesetting notes in LaTeX. It includes both a minimal style for fast compiling and a more colorful and appealing style for writing articles. Multiple user options have been defined to customize the design of the document.
  • Miscellaneous Projects — A repository which contains mini-projects which are not notable enough to have a dedicated repository for themselves.
